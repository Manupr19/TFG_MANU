Aplicación Web sobre comunicación asistida mediante seguimiento ocular

Este proyecto consiste en el desarrollo de una aplicación web funcional que permita a cualquier persona con dificultades en el habla poder efectuarla mediante sus ojos. Para ello, se realizará el conveniente procesado de imágenes, tomadas desde la webcam, mediante algoritmos de aprendizaje automático. además, se realizará un backend y un frontend con tecnologías actuales.
